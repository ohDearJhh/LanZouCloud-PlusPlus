# LanZouCloud++
一个蓝奏云客户端

### 简介
实际上已经有一个蓝奏云的客户端了    
但是还是想开发一个自己使用的客户端  
功能上会按照我自己的需求来写

### 主要功能

可突破官方上传容量限制(不建议使用)    
开启此功能需要承担相应的后果(律师函警告bushi)   
方便快捷的上传与下载文件  
无需登录即可下载分享文件

所以开发进度可能会慢一些
